# looping.html

File ini berisi contoh penggunaan perulangan (looping) di JavaScript.

## Cara Menjalankan

1. Buka file `looping.html` di browser (klik dua kali atau buka dengan aplikasi browser).
2. Hasil perulangan akan langsung tampil di halaman web.

## Penjelasan
- Kode menggunakan perulangan `for` untuk menampilkan angka dari 0 sampai 4.
- Hasil perulangan ditampilkan menggunakan `document.write()`.

## Tujuan
- Memahami dasar penggunaan perulangan di JavaScript.
- Melihat hasil looping langsung di halaman web.
